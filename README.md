# MehrChain

A supportive and mindful companion for your personal growth journey. MehrChain is a Telegram Mini App (TMA) designed to help you commit to positive goals, track your progress, and build empowering habits.

## Status

**[WIP] Work in Progress - Pre-Alpha**

This project is currently in the initial development phase. We are building the core features and defining the user experience.

## Tech Stack

-   **Frontend:** Angular
-   **Backend:** NestJS
-   **Database:** PostgreSQL
-   **Platform:** Telegram Mini App (TMA) & PWA
